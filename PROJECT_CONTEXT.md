# TechBusiness Website - Project Context

## Overview
A modern, professional website for a technology business targeting general consumers. The site showcases comprehensive tech services, expert team members, transparent pricing, and provides easy contact options.

## Brand Identity
- **Name:** TechBusiness
- **Industry:** Technology Solutions
- **Target Audience:** General consumers and businesses seeking technology services
- **Style:** Modern, professional, clean, and trustworthy

## Color Palette
- **Primary:** Blue (#2563EB) - Trust, professionalism, technology
- **Secondary:** Slate (#1E293B) - Sophistication, depth
- **Accent:** White/Gray - Clean, modern, spacious
- **Rationale:** Blue conveys trust and professionalism essential for tech services. Slate adds sophistication while maintaining readability.

## Design Philosophy
1. **Clean & Modern:** Generous white space, large typography, professional imagery
2. **User-Focused:** Clear navigation, intuitive layout, easy contact options
3. **Professional:** High-quality images, consistent branding, polished interactions
4. **Trust-Building:** Team profiles, transparent pricing, security indicators

## Technical Stack
- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Images:** Unsplash (professional tech imagery)

## API Integration
- **Form Submissions:** POST to https://deep-api-server-2moiw.kinsta.app/api/form-submissions
- **Payment Processing:** Stripe checkout link
- **Required Fields:** founder_id, project_id (hidden)
